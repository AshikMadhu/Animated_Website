## ✨ Overview

This project is a **high-end animated landing page** for a mechanical watch, designed with:

- 🎬 Cinematic scroll-based animation  
- ⚙️ Exploded mechanical watch reveal  
- 💡 Dynamic feature storytelling  
- 🎯 Apple-style smooth transitions  
- 🖤 Minimal luxury UI  

---

## 🎥 Key Features

### 🔄 Scroll-Based Animation
- Frame-by-frame animation synced with scroll  
- Smooth transitions using optimized rendering  
- Realistic motion and easing  

### ⚙️ Exploded View Animation
- Watch disassembles into mechanical components  
- Precision-aligned parts (gears, dial, case)  
- Physics-inspired movement  

### 🧠 Smart Content Placement
- Text appears in empty visual space  
- Feature highlights synced with animation  
- Clean, distraction-free UI  

### 🎨 Premium UI Design
- Dark luxury theme (black + neon accents)  
- Cinematic lighting & reflections  
- Minimal, high-end layout  

---

## 🧩 Tech Stack

- ⚛️ HTML / CSS / JavaScript  
- 🎞️ Scroll Animation (GSAP / custom logic)  
- 🖼️ Image Sequence Rendering  
- 🎯 Responsive Design
